Παρατηρήσεις: 
1. Εχουμε προσθεσει default στην incase ενω δεν υπαρχει στην γραμματικη της γλωσσας που μας δινεται στις διαφανειες.
2. Επειδή στη Cimple ο μονος τύπος δεδομένων είναι ο ακέραιος (4 bytes), στις εγγραφές του πίνακα συμβόλων παραλείπουμε αυτή τη παράμετρο διότι εννοείται.
3. Έγινε μια μετατροπή στον μεταφραστή να αναγνωρίζει τις δηλώσεις σταθερών (const). Αυτή η προσθήκη έγινε στον κανόνα declarations.
4. Μονο στην περιπτωση που η συνάρτηση εκτελεστεί μεσω του κανονα callstat --> call ID(actualparlist) δεν μας ενδιαφέρει η τιμή επιστροφής της. Για αυτόν τον λόγο αν η actualparlist ενεργοποιηθεί από την callstat τότε δεν παράγουμε τετράδα par, T_x, ret, _
5. Στο αρχείο test.symb που παράγεται, φαίνεται ο πίνακας συμβόλων σε διαφορετικές φάσεις εισαγωγής και διαγραφής επιπέδων.
